# L15 – Integraler

## Dagordning
* Innebörd samt beräkning med integraler.
* Integreringsregler.

## Mål med lektionen
* Förstå vad integraler är.
* Förstå begreppet primitiv funktion och ta fram denna för enkla funktioner.
* Kunna utföra enkla integreringar.
