# L15 – Integraler

## Dagordning
* Integraler - area och summa.
* Integreringsregler.

## Mål med lektionen
* Förstå vad integraler är.
* Förstå begreppet primitiv funktion och ta fram denna för enkla funktioner.
* Kunna utföra enkla integraler.
